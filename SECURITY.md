# Security Policy

## Supported Versions

Currently only the latest version (v5.2) is supported.

## Reporting a Vulnerability

Email vulnerabilities to [your-email@example.com] or open a private issue.

## Known Limitations

- Designed for trusted networks only
- No DoS protection beyond basic rate limiting
- No forward secrecy (keys persist)

Planned for v6.0: mutual authentication, key rotation, DoS hardening.
