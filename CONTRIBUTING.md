# Contributing to RAZOR-MESH

Thank you for considering contributing to RAZOR-MESH!  
We welcome bug reports, feature requests, documentation improvements, code fixes, tests, and new ideas.

## How to Contribute

1. **Find something to work on**
   - Look at open [Issues](https://github.com/YOUR-USERNAME/razor-mesh/issues)
   - Check the [Roadmap](#roadmap) section in README.md
   - Or propose a new feature/bug fix

2. **Discuss first (recommended for large changes)**
   - Open an issue or comment on an existing one
   - Email sovereignai86@gmail.com if you prefer private discussion

3. **Fork & branch**
   ```bash
   git clone https://github.com/YOUR-USERNAME/razor-mesh.git
   cd razor-mesh
   git checkout -b feature/your-feature-name
